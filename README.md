# Conversor de Idade Canina em FLUTTER

Aplicativo desenvolvido em **Swift** que converte a idade humana para a idade equivalente em anos caninos.

## Projeto
Projeto criado com foco em praticar lógica de programação e desenvolvimento mobile utilizando Swift. O app recebe a idade informada pelo usuário e realiza o cálculo para a idade canina.

## Tecnologias
- Swift / XCode

## Aprendizados
- Manipulação de inputs
- Lógica de conversão de valores
- Estrutura e interface básica de aplicativos iOS
